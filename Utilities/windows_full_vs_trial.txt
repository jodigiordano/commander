Comment générer la version Windows, full et trial.
==================================================

Version Full :
Propriétés TDA > Application > Nom de l'assembly : Commander - World 1
Propriétés TDA > Publier > Options > Description > Nom du produit : Commander - World 1

Version Trial :
Propriétés TDA > Application > Nom de l'assembly : Commander - World 1 - Trial
Propriétés TDA > Publier > Options > Description > Nom du produit : Commander - World 1 - Trial
